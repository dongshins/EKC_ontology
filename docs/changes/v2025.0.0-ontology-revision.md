## 3. Versioning & lineage (OWL best-practice alignment)

### 3.1 Added: owl:versionIRI / owl:priorVersion / owl:versionInfo
* Added in v2025.0.0 header:
  - `owl:versionIRI`   `http://dh.aks.ac.kr/ontologies/ekc-2025`
  - `owl:priorVersion` `https://dh.aks.ac.kr/ontologies/ekc-2022-alpha2507`
  - `owl:versionInfo`  "EKC 2025"@en
* Meaning:
  - base ontology IRI(`…/ekc`)를 기준점으로 두면서, 배포물은 version IRI로 고정해 인용/재현성을 확보하는 구성으로 기록됨.
<!--* Observation (as-is in the TTL):

### 3.2 Added: dcterms:replaces
* Added:
  - `dcterms:replaces` `http://dh.aks.ac.kr/ontologies/ekc-2022-alpha2507`
